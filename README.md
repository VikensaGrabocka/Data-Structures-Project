This project is a Java-based desktop application developed using JavaFX, showcasing the practical implementation of fundamental data structures. It is divided into two independent parts:

1.Doctor Management System (Binary Search Tree)

2.Contacts Management System (Linked List)

Each part demonstrates how core data structures can be applied to solve real-world problems efficiently.


Part 1: Doctor Management System

🔍 Description

This module manages doctor records using a Binary Search Tree (BST). It ensures efficient insertion, deletion, searching, and sorting of doctor data.

⚙️ Implemented Features

➕ Add a doctor

❌ Delete a doctor

🔎 Search for a doctor

📊 Sort doctor records (inherent BST ordering)

🔁 Tree traversals:

Level-order traversal

Pre-order traversal

Post-order traversal

🏗️ Data Structure Used

Binary Search Tree (BST)

📇 Part 2: Contacts Management System

🔍 Description

This module simulates a contact list using a Linked List, allowing dynamic manipulation of contacts and navigation between them.

⚙️ Implemented Features

➕ Insert a contact

❌ Delete a contact

⏭️ Call next contact

⏮️ Call previous contact

🔁 Call again (redial functionality)

🔀 Shuffle contacts

🔄 Traverse:

Forward traversal

Backward traversal

🏗️ Data Structure Used

Linked List

💻 Technologies Used

Java

JavaFX (for GUI development)
